## LAMBDA FUNCTIONS — Explained Simply

A lambda is a small anonymous (no name) function written in one line. Normal function:

```python
def square(x):
    return x * x
```

Same thing as lambda:

```python
square = lambda x: x * x
```

The syntax is: `lambda parameters: expression`

It automatically returns the result — no `return` keyword needed. Lambdas are most useful when you need a quick throwaway function, especially inside `sorted()`, `map()`, `filter()`, etc.

```python
# Sort list of tuples by second element
pairs = [(1, 'b'), (3, 'a'), (2, 'c')]
pairs.sort(key=lambda x: x[1])
# Result: [(3, 'a'), (1, 'b'), (2, 'c')]

# Filter even numbers
nums = [1, 2, 3, 4, 5, 6]
evens = list(filter(lambda x: x % 2 == 0, nums))
# Result: [2, 4, 6]

# Double every number
doubled = list(map(lambda x: x * 2, nums))
# Result: [2, 4, 6, 8, 10, 12]
```

That's it — nothing more complicated than that. Now the questions.

The user is asking about Instagram follow request disappearing.Yeh common issue hai, kuch reasons ho sakte hain:

**Most likely reasons:**

1. **Unhone request decline kar di** — Private account wale instantly decline kar sakte hain, isliye "Requested" gayab ho jaata hai aur wapas "Follow" aa jaata hai

2. **Unhone block kar diya** — Agar block kiya toh profile nahi milegi ya "Follow" button wapas aayega

3. **App glitch** — Sometimes Instagram bug ki wajah se button flicker karta hai, try karo dubara send karo

4. **Unka account deleted/deactivated hua** — Profile disappear ho sakti hai temporarily

---

**Kaise check karein:**

- Unka profile search karo — agar **milta hai** toh request decline hui ya glitch tha
- Agar **nahi milta** — block ho gaye ya account deleted hai
- Agar milta hai toh **dubara Follow** bhejo

## 🧱 Phase 1 — The Foundation (3–6 months)

These are non-negotiables. Bina iske aage kuch nahi hoga.

**Networking Fundamentals**
- TCP/IP, DNS, HTTP/S, FTP, SSH protocols
- How routers, firewalls, and proxies work
- Tool: **Wireshark** (packet analysis)
- Resource: *CompTIA Network+* or free **Professor Messer** videos

**Linux Mastery**
- Command line, file permissions, bash scripting
- Distros to use: **Kali Linux** (hacking), **Ubuntu** (daily use)
- Resource: *OverTheWire: Bandit* (free wargame)

**Programming Basics**
- **Python** — scripting, automation, writing tools (you're already on this ✅)
- **Bash** — for automating attacks/recon
- Basic **JavaScript** — for web hacking (XSS etc.)

---

## 🔍 Phase 2 — Core Ethical Hacking Concepts (4–6 months)

**The 5 Stages of Ethical Hacking:**
1. **Reconnaissance** — passive & active info gathering (Shodan, theHarvester, Maltego)
2. **Scanning** — Nmap, Nessus
3. **Exploitation** — Metasploit Framework
4. **Post Exploitation** — privilege escalation, pivoting
5. **Reporting** — documenting findings professionally

**Key Areas to Study:**
- Web App Security — OWASP Top 10 (SQL Injection, XSS, CSRF, IDOR etc.)
- Network Attacks — MitM, ARP spoofing, sniffing
- Password Attacks — hashcat, john the ripper
- Social Engineering basics

---

## 🧪 Phase 3 — Hands-On Practice (Ongoing)

This is where real learning happens. Labs > theory always.

| Platform | What You Get |
|---|---|
| **TryHackMe** | Beginner-friendly, guided rooms |
| **Hack The Box** | Realistic machines, intermediate+ |
| **PortSwigger Web Academy** | Best free web hacking labs |
| **VulnHub** | Downloadable vulnerable VMs |
| **PicoCTF** | CTF challenges for beginners |

Start with **TryHackMe** → then move to **Hack The Box**.

---

## 📜 Phase 4 — Certifications (The Industry Proof)

| Cert | Level | Cost | Worth It? |
|---|---|---|---|
| **CompTIA Security+** | Beginner | ~$400 | ✅ Entry-level must |
| **eJPT** (eLearnSecurity) | Beginner | ~$200 | ✅ Affordable, practical |
| **CEH** (EC-Council) | Intermediate | ~$1000 | ⚠️ Known but theory-heavy |
| **OSCP** (OffSec) | Advanced | ~$1500 | 🔥 Gold standard, most respected |
| **PNPT** (TCM Security) | Intermediate | ~$400 | ✅ Practical, great value |

For freshers → **eJPT → PNPT → OSCP** is the best path.

---

## 💼 Phase 5 — Specializations (Pick Your Lane)

- 🌐 **Web App Pentesting** — most job openings (Bug Bounty friendly)
- 🔴 **Red Teaming** — advanced adversary simulation
- 🔵 **Blue Team / SOC Analyst** — defense side (easier to get hired first)
- 🔓 **Malware Analysis / Reverse Engineering** — niche, high-paying
- 🐛 **Bug Bounty Hunting** — freelance, platforms like HackerOne, Bugcrowd

---

## 💰 Bug Bounty — Earn While Learning

Bhai ye MERN developer ke liye extra advantage hai — web vulnerabilities samajhna tujhe already easier lagega.

- Start on **HackerOne** or **Bugcrowd**
- Focus on OWASP Top 10 vulns
- Even ₹50k–₹5L+ per valid bug possible

---

## 🗺️ Realistic Timeline

```
Month 1–3   → Networking + Linux + Python
```

## ⚖️ Stay Legal — Always

> Ethical hacking = **permission-based** hacking only. Bina permission ke kisi ka system hack karna crime hai (IT Act 2000, India).

Always use your **own lab**, **CTF platforms**, or **bug bounty programs** with scope defined.

## Quick Fix — Guest Additions Install Kar

Ye sabse important step hai. Bina Guest Additions ke fullscreen properly kaam nahi karta.

**Terminal khol aur run kar:**

```bash
sudo apt update
sudo apt install -y virtualbox-guest-additions-iso
sudo apt install -y build-essential dkms linux-headers-$(uname -r)
```

Phir VirtualBox menu mein ja:
**Devices → Insert Guest Additions CD Image**

CD mount hogi, uske baad:

```bash
sudo /media/$USER/VBox_GAs_*/VBoxLinuxAdditions.run
```

Reboot kar:
```bash
sudo reboot
```

---

## Fullscreen Toggle

Guest Additions install hone ke baad:

```
Host Key + F
```

VirtualBox mein default **Host Key = Right Ctrl**

So press karo: **Right Ctrl + F** → fullscreen!

---

## Agar Bina Guest Additions Ke Karna Hai (Quick)

VirtualBox menu se:
**View → Full-screen Mode** (ya Scaled Mode)

---

**Recommended order:**
1. Guest Additions install kar (upar wala command)
2. Reboot
3. Right Ctrl + F

Screen bilkul fullscreen ho jayegi bhai, resolution bhi auto-adjust ho jaayega! 🖥️

### Terminal Mein
| Action | Shortcut |
|---|---|
| Copy | `Ctrl + Shift + C` |
| Paste | `Ctrl + Shift + V` |

Normal `Ctrl+V` terminal mein **kaam nahi karta** — `Shift` add karna padta hai.

---

### Browser / Normal Apps Mein
Same as Windows:
| Action | Shortcut |
|---|---|
| Copy | `Ctrl + C` |
| Paste | `Ctrl + V` |

---

### 🖱️ Middle Mouse Button (Linux Secret)
Ye Linux ka superpower hai —

1. Koi bhi text **select kar** (click + drag)
2. Jahan paste karna hai wahan **middle mouse button click kar**

Automatically paste ho jaata hai — Ctrl+C bhi nahi dabaana padta! 🔥

---

### VirtualBox Mein Host → VM Paste

Agar Windows se Ubuntu VM mein paste nahi ho raha toh:

**VirtualBox → Devices → Shared Clipboard → Bidirectional**

Enable kar do — phir normally `Ctrl+Shift+V` se terminal mein paste hoga. ✅

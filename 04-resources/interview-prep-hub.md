## General Technical Cheat Sheet

### React
- `useState` — local component state
- `useEffect` — side effects, runs after render
- `useContext` — consume context without prop drilling
- `useRef` — DOM access or persist value without re-render
- `useMemo` / `useCallback` — performance optimization

### Node + Express
- Middleware: `app.use()` runs before route handlers
- `req.params` vs `req.query` vs `req.body`
- Error middleware: 4 args `(err, req, res, next)`
- JWT: sign with secret, verify on protected routes

### MongoDB
- `find()`, `findOne()`, `insertOne()`, `updateOne()`, `deleteOne()`
- Aggregation: `$match` → `$group` → `$project` → `$sort`
- Indexes improve query speed

### REST API Design
- GET (read), POST (create), PUT (replace), PATCH (update), DELETE
- Status codes: 200 OK, 201 Created, 400 Bad Request, 401 Unauthorized, 404 Not Found, 500 Server Error

/daily
  Creates today's daily note at 01-daily/YYYY-MM-DD.md
  Includes: Morning tasks, learning block, job hunt log, wins, blockers
  Auto-chains: links to yesterday's note and tomorrow's note
  Use it: Every morning to plan your day

/inbox
  Ingest any source into the wiki permanently
  After /inbox, paste: a URL / article text / job description / notes
  Creates: summary page in 04-resources/
  Also updates: index.md, log.md, and relevant topic pages
  Use it: Every time you read something worth keeping

/lint
  Vault health check — scans the entire Brain for issues
  Finds: orphan pages, missing frontmatter, broken wikilinks,
         pages not in index.md, stale status fields
  Use it: Once a week to keep the vault clean

================================================================================
  DAILY WORKFLOW (recommended routine)
================================================================================

MORNING (5 min)
  1. Type /daily in Claude → opens today's note
  2. Write your 3 tasks for the day in Obsidian
  3. Check review-queue for anything due today

DURING THE DAY
  - Learn something? → /inbox + paste it
  - Apply to a job? → "Log application to [Company] for [Role]"
  - Finish a study session? → "Add [concept] to review queue"
  - Have a meeting/interview? → "Log a meeting with [person]"
  - Write code you want to keep? → "Save this snippet: [code]"

EVENING (5 min)
  - Open today's daily note in Obsidian
  - Fill in Wins and Blockers sections
  - Write one sentence about what you learned

SUNDAY
  - Type "Do my weekly review" → fills weekly-YYYY-WW.md
  - Check /lint for vault health
